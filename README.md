# Library Management System

A Windows Forms application with Telerik controls for managing a library.

## Features
- Add, edit, and delete books, authors, and customers
- View reports for any list: current books and customer lists
- Search and sorting functionality
- Simple and user-friendly interface

## Technologies
- C# with Windows Forms and Telerik controls
- SQLite database
- User-friendly interface
- Complete and well-structured logic for managing a library
